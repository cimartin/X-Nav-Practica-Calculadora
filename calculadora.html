Instrucciones del teclado:

para introducir un número por teclado presione simplemente su tecla

para sumar presione la tecla "+"

para restar presione la tecla "-"

para mutiplicar presione la tecta "m"

para dividir presione la tecla "d"

para usar una coma decimal puede presionar tanto la tecla del punto "." como la de la coma ","

para obtener el resultado presione "intro"

para borrar el ultimo numero de la pantalla presione la "tecla retroceso"

para borrar el número completo presione la "barra espaciadora"
